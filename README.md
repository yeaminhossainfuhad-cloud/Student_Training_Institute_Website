# Tech Skills Academy Website

## 📖 Project Overview

**Tech Skills Academy** is a multi-section training institute website developed using **HTML only** as part of an academic assignment. The website showcases information about a fictional training institute, including available courses, trainers, class schedules, and a student registration form.

## 🎯 Objective

The purpose of this project is to demonstrate fundamental HTML skills by creating a structured and user-friendly website without using CSS or JavaScript.

## 🛠️ Technologies Used

* HTML5

## 📂 Project Structure

```text
Student_Training_Institute_Website/
│
├── index.html
│
└── asset/
    ├── logo.png

```

## ✨ Features

### 1. Header Section

* Institute logo
* Institute name
* Navigation menu with anchor links:

  * Home
  * Courses
  * Trainers
  * Schedule
  * Admission

### 2. About Institute

* Welcome heading
* Institute description
* Uses HTML formatting tags:

  * `<strong>`
  * `<em>`
  * `<mark>`

### 3. Available Courses

* Unordered list of courses:

  * Web Development
  * Graphic Design
  * Digital Marketing
  * UI/UX Design
  * Data Analysis

* Ordered list of learning steps:

  1. Registration
  2. Admission Test
  3. Enrollment
  4. Training
  5. Certification

### 4. Trainers Section

* Trainer images
* Trainer names
* Designations
* Experience details

### 5. Class Schedule

* HTML table with:

  * Course Name
  * Duration
  * Class Days
  * Class Time

### 6. Student Registration Form

* Full Name (Required)
* Email (Required)
* Phone Number (Required)
* Password (Minimum 8 characters)
* Gender Selection
* Interests Selection
* Course Dropdown
* CV Upload
* Reason for Joining
* Submit Button

### 7. Footer

* Copyright Information
* Social Media Links:

  * Facebook
  * LinkedIn
  * YouTube

## 🚀 How to Run

1. Download or clone this repository.
2. Ensure all images are placed inside the `asset` folder.
3. Open `index.html` in any modern web browser.
